# Portfolio Website – Milestone 1 (Code Institute)

A personal portfolio website created as part of **Code Institute’s Milestone 1 Project**. This site introduces who I am, showcases my early development work, and provides a simple way for visitors to get in touch. It demonstrates core skills in **HTML**, **CSS**, **Bootstrap** and **responsive design**.

---

## 1. UX Strategy

### User Goals
- Learn who I am and what I do.
- View my projects and understand my skill level.
- Navigate the site easily on any device.
- Contact me quickly and simply.

### Site Owner Goals
- Present myself professionally as a junior developer.
- Demonstrate my understanding of front‑end fundamentals.
- Build credibility through clean design and clear content.
- Provide a foundation to expand as my skills grow.

### Target Audience
- Recruiters and employers.
- Code Institute assessors.
- Fellow students and collaborators.
- Anyone interested in my work.

## 6. Deployment

### GitHub Pages Deployment
The site was deployed using GitHub Pages:

1. Go to the repository on GitHub.
2. Navigate to **Settings**.
3. Select **Pages** from the left menu.
4. Under *Source*, choose **main branch**.
5. Save and wait for the site to build.

**Live Site:**  
👉 *https://bradapple.github.io/milestone-1-project-my-portfolio/index.html*

### How to Clone
1. Go to the repository.
2. Click **Code**.
3. Copy the HTTPS/SSH link.
4. Run:  
   `git clone https://github.com/Bradapple/milestone-1-project-my-portfolio.git`

### How to Fork
1. Go to the repository.
2. Click **Fork** in the top-right corner.
3. A copy will be created in your GitHub account.
